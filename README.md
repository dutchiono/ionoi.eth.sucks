# ionoi.eth.sucks

Personal website deployed on GitHub Pages.

## Setup

1. Push this repo to GitHub (already set up at `git@github.com:dutchiono/ionoi.eth.sucks.git`)
2. Go to Settings > Pages in your GitHub repo
3. Enable GitHub Pages (source: GitHub Actions)
4. Your site will be at: `https://dutchiono.github.io/ionoi.eth.sucks`

## Point ENS to GitHub Pages

1. Go to [ENS App](https://app.ens.domains)
2. Connect your wallet
3. Select `ionoi.eth.sucks`
4. Go to "Records" tab
5. Add a "Text Record":
   - Key: `website`
   - Value: `https://dutchiono.github.io/ionoi.eth.sucks`
6. Or set "Content Hash" to point to your GitHub Pages URL

## Deploy

Just push to `main` branch - GitHub Actions will automatically deploy.
